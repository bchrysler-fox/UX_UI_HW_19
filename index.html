My UX Skills

User Research

Brian approaches user research by not only looking at the core audience and getting their perspective, but also getting the perspective of individuals not in the target audience in order to locate a knowledge gap that can lead to a better site.

User Interface Design

Taking the user insights from research, Brian focuses on designing interfaces that make sense for the core users while integrating insights from outsode groups. Additionally, his designs focus on using colors that are culturally enticing in order to communicate intent quickly and clearly.

Web Development

Taking the above insights, Brian creates a basically-coded website to give the coding specialists the goal to work through.
